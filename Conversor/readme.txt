# Conversor de Imágenes Local

Esta es una aplicación web que te permite convertir el formato de tus imágenes de manera rápida, segura y totalmente privada.

## ¿Cómo funciona?

La aplicación funciona completamente en tu navegador. Ninguna de tus imágenes se sube a un servidor, garantizando que tu privacidad nunca se vea comprometida.

## Características

- **Conversión Local:** Todo el proceso se realiza en tu propio dispositivo.
- **Múltiples Formatos:** Convierte tus imágenes a formatos populares como JPEG, PNG, WEBP y GIF.
- **Soporte para HEIC:** ¿Tienes fotos de un iPhone? Puedes convertirlas desde formato HEIC/HEIF sin problemas.
- **Fácil de Usar:** Simplemente arrastra y suelta tus imágenes, elige el formato de salida y descarga un archivo ZIP con todas las imágenes convertidas.
- **Ajuste de Calidad:** Controla el nivel de compresión para los formatos JPEG y WEBP para optimizar el tamaño del archivo.
- **Multi-idioma y Tema Oscuro:** Interfaz disponible en varios idiomas y con un agradable tema oscuro.

## Pasos para usar la aplicación

1.  **Abre el archivo `Conversor.html`** en tu navegador web preferido (como Chrome, Firefox, o Edge).
2.  **Sube tus imágenes:** Arrastra y suelta los archivos en el área indicada o haz clic para seleccionarlos desde tu ordenador.
3.  **Elige el formato de salida:** Selecciona el formato al que deseas convertir tus imágenes (por ejemplo, PNG).
4.  **(Opcional) Ajusta la compresión:** Si eliges JPEG o WEBP, puedes mover el deslizador para ajustar la calidad y el tamaño del archivo.
5.  **Nombra el archivo de descarga:** Escribe un nombre para el archivo ZIP que contendrá tus imágenes.
6.  **Convierte y Descarga:** Haz clic en el botón "Convertir y Descargar". Un archivo `.zip` se guardará en tu carpeta de descargas.
